# MottaCleaner v1.1.0

O MottaCleaner é uma ferramenta simples e eficiente desenvolvida para realizar a limpeza segura de arquivos temporários do sistema Windows.

## Funcionalidades
- **Limpeza Profunda:** Remove arquivos das pastas `TEMP` do usuário e `C:\Windows\Temp`.
- **Transparência Total:** Lista na tela cada arquivo que está sendo removido.
- **Segurança:** Pula automaticamente arquivos que estão sendo usados pelo sistema, evitando erros.
- **Modo Administrador:** Requer privilégios elevados para garantir a limpeza completa.

## Como usar
1. Baixe o executável na aba **Releases** do repositório.
2. Clique com o botão direito no `MottaCleaner.exe`.
3. Selecione **"Executar como administrador"**.
4. Aguarde a conclusão da limpeza.
